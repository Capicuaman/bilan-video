# src/experimental

Learning artifacts from earlier development phases. Not used in production.

## templates/

Audio A/B test variants of QuickTipVideo (test1–test5). Each tried a
different background music track (Flashlight Ambient, Electro Drone,
Dreamy Ambient, 118 BPM, Quick Start). The winner informed the audio
strategy in the production `QuickTipVideo.tsx`.

## components/

`BrandIntro.tsx` and `BrandOutro.tsx` — first-generation brand bookends,
superseded by `BrandIntroPerfected.tsx` and `BrandOutroPerfected.tsx`
in `src/components/`.

Do not import these in `Root.tsx` or production scripts.
